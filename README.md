# Listing Boilerplate

Stack

  - Expo SDK 39 (Fast Deployment)
  - Firebase (Serverless Backend)
  - React-Native

*Status: In Progress*
### Installation

```sh
$ git clone <URL.git>
$ cd listing-boilerplate
$ npm install
$ expo start
$ i for Apple Simulator , a for Android
```
